# 📊 Auditoria Técnica FUNDEF - Município de Matinha (MA)

Este sistema é uma ferramenta de suporte estratégico desenvolvida para o processamento de dados e auditoria tributária do rateio dos precatórios do FUNDEF (1998-2006).

### 🚀 Funcionalidades principais:
- **Motor de Cálculo RRA:** Processamento automatizado de Rendimentos Recebidos Acumuladamente.
- **Regra de Transição IRRF:** Aplicação da lógica tributária vigente, incluindo a parcela adicional a deduzir para faixas de transição.
- **Segregação de Juros:** Proteção integral da parcela indenizatória (Juros de Mora).
- **Compatibilidade SIP:** Estrutura de dados preparada para exportação e importação no Sistema Integrado de Pessoal.

---
**Desenvolvido para garantir segurança jurídica e precisão matemática na gestão pública municipal.**
